# ASR Ecosystem Alignment Pass

This pass aligns the Operational Systems Demo with the current ASR Operating Manual and live ASR website positioning.

## Canonical role

The Operational Demo is **Workflow Proof**. It should demonstrate public intake, workflow states, records, reports, invoices, admin visibility and export-ready operational data.

## Language rules

- Keep the demo practical and public-safe.
- Show workflow value more than menu/navigation novelty.
- Link back to `asrweb.ie` and `asrweb.ie/portfolio` as the primary proof layer.
- Keep this demo positioned as part of the ASR ecosystem, not as an isolated product.
