# Browser & Social Preview Notes

This repo uses shared ASR Web Services browser/social branding.

Assets:
- `/favicon.ico`
- `/img/asr-og-portfolio-mountain-qr.webp`

These are intentionally shared across the ASR ecosystem so browser tabs, link previews and social previews feel consistent with the main ASR website.


Release note: large source PNG logo files are intentionally excluded from the public release package. The runtime uses compressed WebP assets plus the compact favicon.
