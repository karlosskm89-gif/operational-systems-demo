# Visual Proof Notes

For future portfolio screenshots, prefer views that show operational value rather than isolated interface states.

Recommended screenshots:
- Records or operational history views.
- Reporting and export views.
- Workflow/dashboard views that show visibility across the system.
- Invoicing or status views where financial or administrative context is clear.

Avoid relying too heavily on:
- Menus.
- Empty navigation screens.
- Generic forms without operational context.

The Operational Systems Demo should visually demonstrate workflow proof: clearer records, reporting, exports and operational visibility.
